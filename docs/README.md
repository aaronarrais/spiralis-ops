# docs/ — durable knowledge

One file per **doc**: something learned, analyzed, or decided that should be
findable later. If a signal is raw evidence, a doc is the worked-through version:
an analysis, a decision and its rationale, a how-it-works note.

## Frontmatter

```yaml
---
kind: doc
domain: []
status: draft | adopted | superseded
links: []
---
```

## Naming

`<short-kebab-slug>.md` or `<TOPIC>-<YYYY-MM>.md`
